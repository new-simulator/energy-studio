# Energy Studio

## 소개
Energy Plus 기반의 동적 시뮬레이터

### 설치 방법
1. 아래 [Releases](https://github.com/new-simulator/energy-studio/releases) 페이지에서 최신 버전의 `.zip` 파일 다운로드
2. 압축 해제 후 설치 파일(`.exe`) 실행
3. 설치 마법사 안내에 따라 진행

### Artifacts

- Windows Installer (.exe)
- ZIP 파일로 압축된 설치 파일

### Notes

- 본 레포지토리는 납품 및 배포 목적로만 사용됩니다.
- 내부 개발용 히스토리 및 실험 코드는 포함하지 않습니다.
